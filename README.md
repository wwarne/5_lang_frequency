# Frequency Analysis of Words

This script takes a file with text, and prints out top 10 most frequently used words in this text.

# Usage example

```#!bash

$ python lang_frequency.py FAHRENHEIT_451.txt
>
----------------
| Word | Count |
----------------
| the | 2911 |
| and | 1648 |
|  a  | 1078 |
| to  | 872 |
| of  | 869 |
| in  | 818 |
| he  | 816 |
| it  | 609 |
| you | 564 |
| was | 510 |
----------------

```
